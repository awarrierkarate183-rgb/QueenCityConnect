Tools & Technologies Used

All code was written by our team. No pre-built templates or AI-generated code was used.

  • HTML5, CSS3, JavaScript (ES6+) — written from scratch by our team
  • Leaflet.js v1.9.4 — open-source interactive map library
  • OpenStreetMap — open-source map tile data
  • Visual Studio Code — source code editor
  • Live Server (VS Code Extension) — local development preview
  • Vercel — website hosting and deployment
  • Google Fonts — DM Serif Display and DM Sans typefaces
  • Unsplash — royalty-free photography
  • Pexels — royalty-free photography
  • CSS Custom Properties, CSS Grid, CSS Flexbox — layout and theming
  • IntersectionObserver API — scroll-based animations
  • localStorage API — bookmark and dark mode persistence

Compatibility & Testing

The site was manually tested on current desktop and mobile browsers.

  • Desktop compatibility checks completed on Microsoft Edge, Google Chrome, and Mozilla Firefox.
  • Mobile usability checks completed on phone-sized viewports for navigation, card layouts, and buttons.
  • Functional tests completed for search and filter controls, map/list toggle, and form validation.
